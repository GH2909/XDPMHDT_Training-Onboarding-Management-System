Module 1

Giới thiệu HTML

HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc cho các trang web. Nó định nghĩa các phần tử của trang web như tiêu đề, đoạn văn, liên kết, hình ảnh, v.v.

Tại sao HTML quan trọng?

HTML là nền tảng của mọi trang web. Nó cung cấp cấu trúc cho nội dung và làm việc cùng với CSS để tạo kiểu dáng và JavaScript để thêm tính năng interaktif.

Cấu trúc HTML cơ bản

<!DOCTYPE html>
<html>
  <head>
    <title>Tiêu đề trang</title>
  </head>
  <body>
    <h1>Xin chào</h1>
    <p>Đây là nội dung của trang.</p>
  </body>
</html>

Các thẻ HTML thông dụng

  • <h1> đến <h6> - Tiêu đề
  • <p> - Đoạn văn
  • <a> - Liên kết
  • <img> - Hình ảnh
  • <div> - Vùng chứa
  • <button> - Nút bấm
